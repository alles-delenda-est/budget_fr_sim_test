USER ACCEPTANCE TESTS - Budget Simulator v1.9
================================================

These tests cover visual, interactive, and browser-dependent behavior
that automated unit tests cannot validate.


1. CHART RENDERING
------------------
[ ] Debt trajectory chart renders on page load with two lines:
    - Dashed grey "Baseline (PLF 2025)"
    - Solid blue "Scénario"
[ ] Both lines start at the same point (~115.8% debt/GDP)
[ ] X-axis shows years (2025-2035), Y-axis shows "Dette/PIB (%)"
[ ] Maastricht (60%) and alert (100%) reference lines are visible
[ ] Hovering over chart points shows a tooltip with exact values
[ ] Legend is visible and labels match the lines
[ ] Policy impact bar chart appears when any lever is moved off zero
[ ] Bar chart shows three bars: Recettes (green), Dépenses (red), Solde (blue)

2. SLIDER INTERACTIONS
----------------------
[ ] Each slider moves smoothly and updates its displayed value in real time
[ ] Moving any tax slider updates the chart immediately (no lag or flicker)
[ ] Moving any spending slider updates the chart immediately
[ ] Slider value resets to 0 correctly (not stuck at a near-zero float)
[ ] Sliders respect their min/max bounds (cannot drag beyond range):
    - IR: -10 to +10 pp
    - TVA: -5 to +5 pp (step 0.5)
    - IS: -10 to +5 pp
    - Education: -20% to +20%
    - Défense: -15% to +15%
    - Solidarité: -30% to +30%
    - Indexation retraites: -2 to +1 pp (step 0.1)
    - Dépenses santé: -10% to +10%
    - Cotisations: -5 to +5 pp (step 0.5)
    - CSG: -2 to +2 pp (step 0.5)
    - Prime de risque: 0 to 200 bps (step 10)
    - Horizon: 5 to 20 ans

[ ] Clicking "Sarah Knafo" sets all sliders (verify solidarity goes to -60%)
[ ] Clicking "Nouveau Front Populaire" sets all sliders (verify IR goes to +5)
